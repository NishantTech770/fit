
**🚀 About the Project - FitTrack Fitness Application**

FitTrack is a full-stack responsive web application designed to manage and enhance gym services.

The application provides secure authentication and authorization, along with role-based 🎯 dashboards to deliver a personalized experience for different users.

## 

**The platform is structured into three main dashboards:**
 
Admin Dashboard

Trainer Dashboard

Member Dashboard

##

**Member Dashboard Features✨:**
 
  🔐 Secure User Authentication & Authorization
  
  📊 Personalized User Dashboard
  
  🎯 Goal Selection
  
  💳 Plan Selection
  
  🏋️ Exercise Recommendation Based on User Goals
  
  📏 BMI Calculator
  
  💰 Payment Intergration
  
  👤 Personal Profile Management
  
  🔒 Access Control for Recommended Exercises

##

**Trainer Dashboard Features✨:**
 
  👀 View all gym member
  
  👤 Personal profile management

##

**Admin Dashboard Features✨:**

 👀 View All Trainers 

 👥 View All Members 
 
 ➕ Add New Trainers 
 
 👤 Profile Management 
 
 📊 Dashboard Analytics (Total trainers, members and active plans) 

##

 **✨ Features for Unauthenticated Members-** 
 
 🏋️ View Gym Services

 ℹ️ Learn About the Gym

**How to run the project locally**

Make sure you have Installed these tools in your VS Code to run the project locally

󠁯•󠁏󠁏 Node.js

•󠁏󠁏 Git

•󠁏󠁏 NPM

•󠁏󠁏 VS Code

##

**Step 1 - Clone the Repository**

Command - git clone https://github.com/amankumawat06/Fitness-Application.git

⿻ Copy the above command and 📌paste it on your VS Code Terminal inside an empty folder

##

**Step 2 - Install Dependencies**

Run command in Terminal - npm install

⚠️Make sure you are inside the FitTrack/Empty folder

##

**Step 3 - Replace API path** 

Replace every API path -

*https://fitness-application-three.vercel.app* 

with 

*http://localhost:8080*

##

**Step 4 - Start the Frontend Server**

📂Go to the frontend folder -

  ▶️ Run command - cd frontend

Run the frontend Server -

▶️ Run command - npm run dev

##

⚙️ **Step 5 - Start the Backend Server**

↩️ Go back to the root folder-

 Run command - cd ..

📂 Go to the backend folder- 

  ▶️ Run command - cd backend

  Run the backend server - npm start

##

**Step 6 - Open in browser🌐**

Visit - http://localhost:5173/

🎉Your project should now be running locally
